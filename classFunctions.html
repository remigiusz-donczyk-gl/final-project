◆ countSetBits()

Functions::countSetBits ( int  $n)
Parameters
[in]$nThe integer to count bits in.
Returns
seen The number of bits set to 1.

Counts set bits in an integer, this is how I have decided to store the memes already seen by a user; each bit represents the 'seen' state of the meme at the same index in the database, this allows for a space-efficient storing method, where each bit stores one meme, instead of a byte (or multiple) in the case it was stored simply as a list of seen images.
